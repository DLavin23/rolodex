# Rolodex

Styleguide gem for Belly

## Use with Middleman

Add to your `Gemfile`:

```
gem 'rolodex', :git => 'https://github.com/bellycard/rolodex.git'
```

Import modules to your SCSS files:

```
@import 'main';
```
